Organize market, macro, crypto, filings, charts, tables, and watchlist context without notebook sprawl.

Reduce local setup debt when the team needs adoption, console access, and operational support.

Treat OpenBB-style access as a finance data layer that can feed internal apps, dashboards, or agents.

Package approved finance sources and parameters before an AI workflow uses them.

## Repository-aware evaluation

Separate data access and workflow operations from investment advice or guaranteed market outcomes.
