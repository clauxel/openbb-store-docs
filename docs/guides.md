# Guides - OpenBB Launch

OpenBB Launch provides a hosted launch path around OpenBB-style market, macro, crypto, filings, API, chart, and agent-ready finance workflows.

## Inspect the OpenBB repo

Review install paths, release activity, examples, docs, and the line between open-source and workspace features.

## Choose the first data surface

Start with one workflow and the exact market, macro, crypto, filings, or internal data it needs.

## Avoid financial-advice confusion

Label outputs as research workflow artifacts and keep human review for decisions.

## Pricing and Support

Plans cover provisioning, workspace access, API-layer operations, team support, and managed launch help rather than raw financial advice.
