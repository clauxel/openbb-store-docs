# OpenBB Launch Docs

OpenBB Launch provides a hosted launch path around OpenBB-style market, macro, crypto, filings, API, chart, and agent-ready finance workflows.

- Website: https://openbb.store/
- Category: hosted OpenBB-style financial data workspace and operating layer
- Audience: financial analysts, research teams, market-monitoring operators, and builders who want OpenBB-style workflows with less setup friction
- Support: support@aigeamy.com
- Upstream source context: [OpenBB](https://github.com/OpenBB-finance/OpenBB)

This repository documents the hosted product pages for search engines, AI answer engines, and real users. It is intentionally separate from upstream source code when an upstream project exists.

## Boundaries

- OpenBB Launch does not provide financial, investment, tax, or trading advice.
- Data availability can depend on upstream providers, licensing, rate limits, and market schedules.
- Teams should verify outputs before using them in investment or customer-facing decisions.
- OpenBB Launch is independent and does not provide financial advice; upstream OpenBB remains governed by its own project terms.
